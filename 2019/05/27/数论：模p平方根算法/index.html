数论 模p平方根算法（C）


模p平方根算法用来快速计算二次同余式

计算模p平方根例子:
$x^2$ $\equiv$ 4 (mod 281)
该二次同余式解为 x $\equiv$ 132 (mod 281)

int  power(int x,int y,int m)//计算 x^y%m
&#123;
    if(y==0)return 1%m;
    int  ret=x%m;
    for(int i=0;i<y-1;i++)
        ret=(ret*x)%m;
    return ret;

    int ret=1;
    for(int i=0;i<b;i++)
        ret*=a;
